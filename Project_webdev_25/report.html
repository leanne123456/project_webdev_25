Project Report

Introduction

This project is a personal portfolio webpage designed to showcase my background, skills, and aspirations in an engaging and structured format. The website was developed using HTML, CSS, and Bootstrap to ensure responsiveness and a modern aesthetic. It meets all the required project guidelines, including:

  • A colored background that adapts to different screen sizes.
  • Well-organized sections for clear navigation.
  • A responsive table to display categorized information.
  • Deployment using GitHub Pages.

Design Process

Before development, I created a boxified wireframe on paper to outline the website structure. This helped define key sections:

  • Header with my name and profile picture.
  • Content areas like "About Me", "Skills", "Education", and "Employment".
  • A table for categorized information.
  • A footer linking to my GitHub.

I then digitized this design and embedded it under "My Design Process" on my portfolio page.

Implementation

1. HTML Structure

The webpage consists of structured sections using Bootstrap's grid system:

  • Navigation Bar: Allows seamless navigation between pages.
  • Header: Includes my name, job title, and profile image.
  • About Me: A full-width section providing a short introduction.
  • Skills & Interests: A structured table displaying my skills.
  • Education & Employment: Neatly formatted lists of my background.
  • Achievements: Showcasing important milestones.
  • Reflection & Future Plans: My thoughts on the project and potential improvements.
  • Footer: Includes a GitHub link and navigation back to the homepage.

2. CSS & Responsiveness

The design uses Bootstrap and custom CSS for styling. Key features include:

  • Background color change based on screen size (white for screens smaller than 460px).
  • Header is centered and balanced, with a properly aligned profile image.
  • All elements are fully responsive, ensuring mobile-friendliness.
  • The employment section was adjusted for better readability.

3. Bootstrap & Table Integration

Bootstrap classes are used throughout the page for spacing and layout. The table meets project requirements:

  • Contains at least three rows and two columns.
  • Displays structured, categorized information.
  • Fully responsive, adapting to smaller screens.

Media Queries & Adaptive Design

The website's CSS media queries ensure it adjusts based on screen width:

  • For small screens (less than 460px), the background changes to white for readability.
  • All sections stack vertically, ensuring no content overlap.
  • Tables and images resize dynamically to fit the screen.

GitHub Deployment

The project was deployed using GitHub Pages, following these steps:

  1. Created a GitHub repository named project_webdev_25.
  2. Pushed all HTML, CSS, and images to the repository.
  3. Enabled GitHub Pages in the repository settings.
  4. Ensured the live site link worked properly before submission.

Challenges & Solutions

  • Header Image Alignment: Adjusted CSS to balance text and image positioning.
  • Table Responsiveness: Used Bootstrap classes to prevent content overflow.
  • Employment Section Formatting: Improved layout for readability.

Reflection & Future Plans

This project allowed me to apply Bootstrap, CSS media queries, and GitHub deployment. If I were to expand it, I would:

  • Add animations and hover effects for a more interactive experience.
  • Implement a contact form to improve engagement.
  • Enhance the design with advanced CSS techniques.
  • Use JavaScript for additional interactivity.

Overall, this project provided valuable experience in responsive web design and deployment.
